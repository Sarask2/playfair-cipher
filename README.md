# 🔐 Playfair Cipher

<img width="1055" height="1169" alt="preview png" src="https://github.com/user-attachments/assets/21e0b899-ff0e-45d1-b126-0b2ffa01f211" />

An interactive web application that demonstrates the Playfair Cipher encryption algorithm. The application allows users to encrypt and decrypt messages while visualizing the generated 5×5 key matrix and the transformation of each letter pair.

## 🌐 Live Demo

https://sarask2.github.io/playfair-cipher/

---

## Features

- Encrypt and decrypt text using the Playfair Cipher.
- Automatically generates the 5×5 key matrix from a keyword.
- Highlights the keyword letters inside the matrix.
- Displays each digraph and the rule applied during encryption or decryption.
- Copy the final output with one click.
- Clean and responsive user interface.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## How to Use

1. Enter a keyword.
2. Type the plaintext (or ciphertext for decryption).
3. Choose **Encrypt** or **Decrypt**.
4. Click **Run Cipher**.
5. View the generated key matrix, output, and step-by-step digraph breakdown.

## Project Purpose

This project was developed as part of the **CS 320 – Computer Security** course to demonstrate the implementation of the Playfair Cipher and provide a simple visual explanation of how the algorithm works.
